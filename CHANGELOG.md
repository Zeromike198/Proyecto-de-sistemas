## Cómo usar este archivo
- Cada vez que se haga un cambio importante en el manual, debe abrirse una rama y un pull request.  
- Cuando el PR sea aprobado y fusionado, se actualiza este archivo.  
- Solo se añade una nueva versión cuando haya cambios suficientes para justificarla.  
- Las fechas deben agregarse en formato: **DD/MM/AAAA** o el que el profesor prefiera.
